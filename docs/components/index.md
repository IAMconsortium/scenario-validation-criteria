The criteria definitions contain the following components:

* **[Types](types.md)** — Distinguishes criteria based on the purpose
* **[Descriptions](descriptions.md)** — Explains the justification for defining criteria and the specific threshold level chosen
* **[Thresholds](thresholds.md)** — Defines threshold levels for all criteria across different years
* **[Reference data](reference_data.md)** — Used for setting threshold levels relative to some external data source
* **[Sources](sources.md)** — Bibliographic information on sources and references used

Use the navigation sidebar or the links above to inspect individual components.
