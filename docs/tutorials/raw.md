# Raw input data
The input data files are located in the `inst/extdata/` subdirectory of the git repository, which is [managed via GitHub](https://github.com/IAMconsortium/scenario-validation-criteria/).

You can clone the full repository by running:
```bash
git clone https://github.com/IAMconsortium/scenario-validation-criteria.git
```

Alternatively, you can download individual files from `inst/extdata/` directory in the latest version of `main` branch [here](https://github.com/IAMconsortium/scenario-validation-criteria/tree/main/inst/extdata).
